# N°1 Suma de pares positivos
```python
suma = 0  

for i in range(5):
    numero = int(input(f"Ingrese el número {i+1}: "))
    
    
    if numero > 0 and numero % 2 == 0:
        print(f"El número {numero} es positivo y par")
        suma += numero

print(f"\nLa suma total de los números positivos y pares es: {suma}")

```
